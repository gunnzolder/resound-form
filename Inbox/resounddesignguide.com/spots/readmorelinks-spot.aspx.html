Readmorelinks-spot

The Readmore links spot is used to make small teaser spots - without an image background - that a visitor can click. The visitor is redirected to the wanted destination.

It consists of a:

  • Header
  • Text
  • Linklist

You can add any number of links to the link list.

It is possible to create a light or a dark version of the spot. 

 

Example code

<div class="spot">
	<div class="context readmoreBox"> 
		<div class="textContainer">
			<h2>
				ReSound Alera®. It simply 
				sounds better
			</h2>
			<div class="mainText">
				The sound processing 
				strategies of ReSound Alera 
				effectively mimic those of 
				the human ear in order to 
				deliver a crisper, cleaner 
				sound and a more natural 
				sound experience.
			</div>
			<ul class="linkList">
				<li>
					<a href="link">
						Your hearing
					</a>
				</li>
				<li>
					<a href="link">
						ReSound
					</a>
				</li>
			</ul>				
		</div>
	</div>
</div>

Dark version has the class 'black' added.

 

Example code

<div class="spot">
	<div class="context readmoreBox black">
