Fact-spot

LIVE demo

The Fact spot is used to make small teaser spots - without an image background - that a visitor can click. The visitor is redirected to the wanted destination.

It consists of a:

  • Header
  • Text
  • Link

It is possible to create a light or a dark version of the spot. When hovering over the spot, it has a mouseover effect demonstrating that the entire spot is clickable.

 

Example code

<div class="spot">
	<div class="context fact">	
		<a href="link">
			<div class="textContainer">
				<h2>
					ReSound Alera®.  It simply 
					sounds better;
				</h2>
				<div class="mainText">
					ReSound Alera® is built on 
					an exciting new platform 
					that delivers sophisticated 
					technologies.<br> <br> The 
					sound processing strategies 
					of ReSound Alera 
					effectively mimic those of 
					the human ear in order to 
					deliver a crisper, cleaner 
					sound and a more natural 
					sound experience.
				</div>
				<div class="clearfix">
					<span class="link">My link</span>
				</div>
			</div>
		</a>	
	</div>
</div>

Dark version has the class 'black' added.

 

Example code
