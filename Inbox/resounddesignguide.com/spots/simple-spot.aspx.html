Simple-spot

LIVE demo

The simple spot is used to make small teaser spots on product pages. 

 

It consists of an:

  • Image / video
  • Header
  • Text
  • Link
  • See more

The image will always be 100 %; be therefore careful not to upload larger images than needed, since large images slow down site performance. When using a video there will be an image with a play button. When clicked a light box will open and the video will play.

 

The simple spot has 6 layouts:

  • 100% size, with image to the left and text under header
  • 100% size, with image to the right and text under header
  • (simple story) 100% size, with image to the left and text beside header
  • (simple story) 100% size, with image to the right and text beside header
  • 1/3 size
  • 1/2 size

The code is almost identical, as seen below:

 

Example code - 100% size, with image to the left

<div class="spot g-100 showSpot">		
	<div class="context boxModel g100 clearfix">				
		<div class="img">
			<img src="image" alt="">
		</div>
		<div class="text">
			<div class="textContainer">
				<h3>ReSound Unite Mini Microphone</h3>
				<div class="textWrapper">
					With ReSound Unite™ Mini Microphone, 
					you can hear speech clearly from 
					up to 7 meters away. Place the 
					microphone close to the person you 
					wish to hear and it will stream 
					speech directly to your hearing aids.
				</div>
			</div>
		</div>
	</div>
</div>


Example code - 100% size, with image to the right

<div class="spot g-100 showSpot">		
	<div class="context boxModel g100 clearfix">				
		<div class="img right">


Example code - (simple story) 100% size, with image to the left and text beside header

<div class="spot g-100 story">		
	<div class="context boxModel g100 clearfix">				
		<div class="img">


Example code - (simple story) 100% size, with image to the right and text beside header

<div class="spot g-100 story">		
	<div class="context boxModel g100 clearfix">				
		<div class="img right">


Example code - 1/3 size

	<div class="context boxModel g33 clearfix">
