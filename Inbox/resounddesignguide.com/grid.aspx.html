Grid

The base grid is a universal grid, divided into 24 vertical units. The grid is measured in percentages, to enable responsive layout and fluid grid.

Surrounding the grid is a wrapper with a maximum width of 1008px, which centers the entire website.

To keep px-margins between the fluid grids, one must add an extra 'div' inside the last grid, with a class of context. It is not possible to add e.i. 'Grid context grid'. It always has to be 'grid grid context'.

The context class will add a margin of 8px on both sides and a margin-bottom of 16px. This will also add a box-shadow and white border around the element.
